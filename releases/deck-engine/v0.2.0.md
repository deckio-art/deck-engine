# @deckio/deck-engine v0.2.0

**Released:** 2026-03-13

## What changed

### Added
- **Funky Punk theme** (`funky-punk`) — a loud, neon, rebellious theme featuring:
  - Hot pink (`#ff2d6f`) + electric lime (`#e0ff00`) + cyan (`#00e5ff`) + purple (`#bf00ff`) on jet-black
  - Bebas Neue headings (uppercase, with chromatic text-shadows)
  - Space Mono for code blocks
  - Sharp edges (near-zero border radius)
  - Scanline overlay texture
  - Wavy underline links
  - Punk bullet markers (✘)
  - Neon gradient horizontal rules
  - Fast, snappy transitions

## Upgrade guide

Use in your deck config or CSS import:
```js
// deck.config.js
export default { theme: 'funky-punk' }
```
```css
@import '@deckio/deck-engine/themes/funky-punk.css';
```
