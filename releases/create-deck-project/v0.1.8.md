# @deckio/create-deck-project v0.1.8

**Released:** 2026-03-13

## What changed

### Fixed
- Remove premature "Project scaffolded" message — the spinner now stays active through all bootstrapping steps instead of stopping after file creation.
- Fix spinner freeze during `npm install` — replace blocking `execSync` with async `exec` (promisified) so the @clack spinner animates correctly while dependencies are being installed.
- Show a clear "Installing dependencies..." stage message before the slow `npm install` step so users always know what is happening.
- Non-fatal engine initialization errors no longer halt the spinner; the project completes successfully and is still usable.

### Changed
- Shorten TUI prompt labels ("Deck title" → "Title", "Deck icon" → "Icon") and tighten design-system option hints for a cleaner interactive experience.
- The bootstrapping flow now uses a single continuous spinner with `s.message()` stage updates instead of stop-and-restart pairs.

## Upgrade guide

No action required. Run `npx @deckio/create-deck-project` as usual.
