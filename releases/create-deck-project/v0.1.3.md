# @deckio/create-deck-project v0.1.3

**Released:** 2026-03-13

## What changed

### Fixed
- Update fallback `@deckio/deck-engine` dependency from `^1.8.2` to `^0.1.1`. Scaffolded projects were pulling the old `1.8.2` version which is missing the `tailwindPlugin` export, causing `SyntaxError` on `vite dev`.

## Upgrade guide

No action required. New projects will now correctly depend on `@deckio/deck-engine@^0.1.1`.
